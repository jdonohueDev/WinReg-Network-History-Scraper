Description:

Before running this script, launch your favorite python interpreter in administrator mode on a 
Windows device which supports Python 3 or greater. To run this script, either copy and paste 
the code or run the printNets.py file directly in your chosen IDE. This script will access your 
Windows Registry directly and return the network name and respective MAC address associated
with your device for all Wi-Fi networks you have accessed which are still
stored within the registry.
